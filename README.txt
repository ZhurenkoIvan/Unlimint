Приложение по конвертации входящих данных в формате .json и .csv.
Создано на основе сброщика проектов Maven с применением фреймворка Spring.
После работы приложения пользователь на выход в консоль получает
конвертированные данные в формате .json.
Конвертирование происходит построчно. Если в строке входных данных содержатся ошибки,
в столбце result будет описана проблема. Если входные данные не содержат ошибок,
то в стоблце result будет статус OK.
Для корректной работы приложения файлы для конвертации нужно поместить в папку, где
лежит jar-файл. Иначе на выходе будет ошибка Файл не найден.
Каждый файл читается отдельным потоком.
Конвертация данных происходит отдельным потоком для каждой строки файла.
Добавление новых форматов возможно созданием дополнительного бина с обработкой этого формата
и добавлением этого типа данных в ENUM FileType.
